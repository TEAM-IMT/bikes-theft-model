# Pasos de instalacion y modificacion del proyecto.
Esta guia presenta los pasos para instalar y ligar un repositorio a los archivos locales, con la finalidad de hacer actualizaciones.

## Verificacion de instalacion y configuracion inicial.
Una vez instalado git, se puede comprobar y modificar importantes parametros con el siguiente codigo:

```
git --version
git config --global user.name "Your Name"
git config --global user.email <your email address>
git config --list
```

## Instalacion de repositorio y link con cuenta
Para *linkear* un repositorio basta con clonar el repositorio en la carpeta local deseada, apuntando a `directory` y ejecutando un `git clone`.

```
git clone https://github.com/Johansmm/bikes-theft-model.git # Credenciales necesarias si el directorio es privado!
cd bikes-theft-model
```

## Modificacion del repositorio
Una vez realizada la configuracion del repositorio, puede ser visto su estado (1), agregar todas las modificaciones hechas (2) para luego actualizarlas localmente (3), con la finalidad de subirlas al repositorio (4) y obtener actualizaciones de otros colaboradores (5 )con los comandos:
```
git status
git add .
git commit -am "Your comment or explanation of update"
git push
git pull
```

Para mayor informacion, consultar `Git-commands.pdf`, instrucciones tomadas del curso de coursera [Server-side Development with NodeJS, Express and MongoDB](https://www.coursera.org/learn/server-side-nodejs/supplement/z2xH0/setting-up-your-development-environment-git-additional-resources) y la web.
